# Boundra v0.1.0

## Release Summary

`boundra@0.1.0`은 workspace 루트가 실수로 npm에 게시된 저장소
스냅샷입니다. 지원되는 runtime export와 CLI entry point가 없으므로
정상 릴리스로 사용하면 안 됩니다.

## Highlights

- 공식 기능 릴리스가 아닙니다.
- npm 패키지 이름을 선점했지만 사용자용 실행 표면은 포함하지
  못했습니다.

## Installation

이 버전은 설치하지 마세요. 이미 설치했다면 첫 정상 preview로
업그레이드하세요.

```bash
pnpm add boundra@^0.1.1 zod
```

## Breaking Changes

- `0.1.0`은 지원 대상에서 제외됩니다.
- 정상 package export를 사용하려면 `0.1.1` 이상으로 업그레이드해야
  합니다.

## Known Limitations

- `exports`가 없습니다.
- `bin` entry가 없습니다.
- workspace source, CI 설정, 문서와 내부 파일이 tarball에 포함됩니다.

## Verification

- 공개 npm tarball을 직접 검사해 105개 저장소 파일이 포함된 것을
  확인했습니다.
- runtime import와 CLI 실행 경로가 없음을 확인했습니다.
